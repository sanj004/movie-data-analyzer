# 🎬 Movie Data Analyzer

Exploratory data analysis on 5000 TMDB movies using Pandas and NumPy.

## 📌 About
This project analyzes a dataset of 5000 movies to uncover insights about profit, ROI, ratings, and genres — using only Pandas and NumPy (no sklearn).

## 🔍 What's covered
- Loading and inspecting real-world movie data
- Cleaning missing values and duplicates
- Filtering movies by runtime and rating
- Feature engineering: profit and ROI calculations
- Statistical analysis using NumPy (mean, median, std)
- Top 10 most profitable and highest rated movies
- Genre-wise average rating and ROI breakdown
- Bar chart visualization of average rating by genre

## 📊 Sample Visualization
Average movie rating by genre plotted as a bar chart using Matplotlib.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- VS Code + Jupyter Extension

## 📂 Dataset
[TMDB 5000 Movie Dataset](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)
